# 📚 Laboratory Works in C

This folder contains laboratory works completed during the 1st year of study in the **C programming language**.

## 📖 Overview

**Semester 1** was dedicated to mastering the fundamentals of programming: branching operators, loops, one-dimensional and multidimensional arrays, dynamic memory allocation, sorting algorithms, and working with strings and function pointers. The main goal was to build a solid understanding of basic algorithms and data structures.

**Semester 2** focused on more advanced topics: binary and text file processing, custom data types (structures, unions, bit fields), linked lists (stacks, queues), and search algorithms using binary trees and hash tables. The emphasis was on efficient data organization, memory management, and algorithmic optimization.

---

Together, these laboratory works represent my journey from writing simple console programs to implementing complex data structures and file-based applications in C.
